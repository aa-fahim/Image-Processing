to create spatial_result.png, spatial_filter("baboon.png", "h.txt") was called
spatial_result.png is the output

for this particular example, a blurring spatial mask was used;
it can be seen that the output image is indeed a blurred version of the input;
you can change h to any spatial mask you'd like of any size M x N;
M and N must be of a odd integer;
keep format the same by entering ',' and space after every value in the mask;
for example "5, 6, 9, 10, 11";
enter every new row of the mask in the next line under the previous line with no space;
